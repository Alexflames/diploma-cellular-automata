\maketitle

Общая постановка задачи: исследовать возможности оптимизации генетического алгоритма для поиска двумерных клеточных автоматов первого порядка. 
Для решения этой задачи необходимо:
\begin{enumerate}
\item разработать гибкое приложение, моделирующую работу клеточных автоматов с возможностями конфигурации генетического алгоритма;
\item добавить визуализацию клеточных автоматов и сбор статистики по экспериментам;
\item провести множество экспериментов с различными конфигурациями и целевыми изображениями;
\item сделать выводы, определить лучшие параметры генетического алгоритма, предложить возможные оптимизации.
\end{enumerate}

В теоретической части работы необходимо описать используемые алгоритмы и их оптимизации, технологии и привести все необходимые предварительные сведения для практической реализации. В экспериментальной  части работы необходимо описать разработанный программный продукт, произвести запуски в различных конфигурациях и сделать соответствующие выводы.
